# Ukraine Auto Customs Admin

## Description
Ukraine Auto Customs Admin is a content and analytics platform for legal analysts who monitor vehicle customs regulations in Ukraine.
It helps teams manage legislative updates, verify official sources, and track user engagement with legal news content.

## Tech Stack
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Node.js / npm

## Features
- Legislative content management for customs updates (`laws.ts`-driven workflow)
- Legal source validation (URL health check for official links)
- Admin dashboard with engagement analytics (clicks, interest rate, reliability indicators)
- Built-in admin onboarding elements (tooltips, contextual explanations, handbook links)
- SEO-ready article architecture (metadata, structured data, sitemap, pagination)

## Setup Instructions
1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Start in production mode:

```bash
npm run build
npm start
```
